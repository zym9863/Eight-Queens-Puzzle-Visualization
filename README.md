# 八皇后问题可视化

[中文](README.md) | [English](README_EN.md)

一个优雅的八皇后问题可视化求解工具，通过现代化的网页界面展示八皇后问题的求解过程。

## 项目简介

八皇后问题是一个经典的回溯算法问题，要求在8×8的国际象棋棋盘上放置八个皇后，使得任何两个皇后都不能互相攻击。本项目通过直观的可视化界面，展示了问题的求解过程，让用户能够更好地理解回溯算法的工作原理。

## 功能特点

- 🎯 实时可视化：动态展示回溯算法的求解过程
- 🎮 交互控制：支持开始求解和重置功能
- 📊 状态展示：实时显示当前尝试次数和已找到的解法数量
- 💫 优雅动画：流畅的皇后放置动画和状态转换效果
- 🎨 现代界面：采用科技感设计风格，提供良好的视觉体验

## 技术实现

- 前端界面：使用HTML5 + CSS3构建响应式布局
- 视觉效果：采用现代CSS特性实现渐变、动画和毛玻璃效果
- 交互逻辑：使用原生JavaScript实现回溯算法和动态渲染
- 性能优化：通过异步操作和延迟控制实现流畅的动画效果

## 使用说明

1. 打开页面后，您会看到一个8×8的国际象棋棋盘和控制面板
2. 点击"开始求解"按钮，程序将开始寻找八皇后问题的解法
3. 观察棋盘上皇后的放置过程，了解回溯算法的工作方式
4. 右侧面板实时显示当前已找到的解法数量和尝试次数

## 界面预览

- 科技感十足的UI设计
- 流畅的动画过渡效果
- 清晰的状态信息展示
- 直观的皇后放置可视化

## 开发亮点

1. 采用CSS Grid布局实现棋盘的完美对齐
2. 使用CSS自定义属性实现主题色统一管理
3. 通过SVG实现皇后图标的高清显示
4. 结合Promise和异步函数实现动画控制
5. 响应式设计确保在不同设备上的良好显示效果

## 本地运行

1. 克隆或下载项目文件
2. 使用现代浏览器打开index.html文件
3. 开始体验八皇后问题的可视化求解过程
